== Description ==

**Add to Quote For Woocommerce** is a powerful tool designed to enhance the functionality of your WooCommerce store by enabling customers to request quotes for products instead of making immediate purchases. With this plugin, customers can add products to a quote cart, review their selections, and submit a quote request for a personalized price or more information.
